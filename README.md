# 数据 Agent

这是通用问数 Agent 平台的代码仓，当前先承接“经营问数”这一标准场景，后续逐步扩展到任意数据源、任意业务主题的通用问数平台。

## 当前定位

- 先打通经营问数最小闭环
- 先实现通用接入、语义建模、问数执行和口径治理的骨架
- 后续再扩展多租户、多客户、多主题复用能力

## 目录说明

- `backend/`：后端服务代码
- `frontend/`：前端页面代码
- `data/`：Doris 建模、初始化数据、质量规则
- `deploy/`：部署脚本、容器、环境配置
- `docs/`：架构、接口、运行手册、测试样例
- `scripts/`：本地工具脚本

## 第一阶段目标

1. 建好工程骨架
2. 接入客户源表到 Doris ODS
3. 形成最小 DIM / ADS
4. 跑通 Wren 首批模型
5. 打通问数请求的最小闭环
